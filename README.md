# Online Exam Cheating Detection Using ANN

## Project Overview
This project uses Artificial Neural Networks (ANN) to detect suspicious behavior during online examinations.

## Technologies Used
- Python
- TensorFlow
- NumPy
- Pandas
- ANN

## Features
- Student behavior analysis
- Data preprocessing
- Classification of suspicious and normal activities
- Deep learning-based prediction
